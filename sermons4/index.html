This site provides an archive of the various sermons and classes of the Mid-County Church of Christ. This audio archive is intended to be a resource for those interested in learning more about God, his Son, and His purpose for our lives.

Currently this site is a work-in-process, and it will take much more time to continue to add audio and other resources. At present, you may find the following:

  • A Master Index of all sermons and classes with limited audio. Use this if you want to search the entire catalogue. If no audio file has been uploaded for the desired items you may contact the tape librarian with a request for the audio.
  • An Abbreviated Index which shows only items in the Master Index which have an audio file
    •
  • A list of Foundational Sermons, or impactful sermons. These are also in the Master Index but have been set-aside as a good introductory place to start
    •
  • A list of sermons annotated in video
